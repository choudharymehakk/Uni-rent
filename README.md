# 🚀 UniRent - Smart Rental Platform

## 📌 Overview
UniRent is a web-based platform that allows users to rent out their unused items and earn money. It simplifies the process of listing, managing, and renting products within a controlled system.

---

## 🎯 Problem Statement
Many people have items that are rarely used but could be useful to others. UniRent provides a platform where users can monetize these items instead of letting them go unused.

---

## ⚙️ Features
- 🔐 User authentication and role-based access (Admin, User)
- 📦 Product upload and management
- 📄 Rental request system
- 📊 Order status tracking
- 🔄 Automated request handling (no manual coordination)

---

## 🛠️ Tech Stack
- **Backend:** Django, Django REST Framework  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL  
- **Tools:** Git, GitHub, VS Code  

---
---

## 🚀 How It Works
1. User signs up and logs in  
2. Users can upload items they want to rent out  
3. Other users can request to rent those items  
4. Admin manages and approves requests  
5. Order status is updated accordingly
